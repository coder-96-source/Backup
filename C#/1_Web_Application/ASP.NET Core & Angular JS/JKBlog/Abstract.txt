프로젝트 주제: ASP.NET Core 2.0 및 Angular JS를 사용하여 블로그 만들기.
프로젝트 이름: JKBlog
기능&목표:
*General
1.Bootstrap Snippet Pack & Glyhfriend 2017을 사용해 UI를 구성.
2.Azure Let's Encrypt를 사용하여 SSL 보안 제공.
3.ASP.NET Core의 Bower를 사용하여 Package 관리

*Home
1.Angular JS를 사용하여 Home, About, Contact에서 reload없이 웹페이지를 이용 가능 하도록 한다.(SPA)
또한 Home에서 Article 클릭시, Pop-up을 통해 reload 없이 Article을 보여준다.
(Learn more 등 navigation bar에서 Topic 클릭시에는 새로운 웹페이지를 로드한다.)

*Article
1.DISQUS Package를 이용하여 사용자 가입없이 댓글을 관리한다.
2.CKEditer Package를 이용하여 글을 쓸 수 있게 한다.
3.Admin과 User 구분하여, User 대상으로 글을 작성할 수 있게 해준다.

*Admin
1.Admin Page는 SB Admin v2.0 Package를 이용하여 UI를 구성한다.

*Database
1.Trigger를 이용하여 삭제된 데이터 백업하기
2.SP, Trigger등을 이용한 향상된 DB 관리
